Para poner Django
mkdir mi_primer_proyecto_django
cd mi_primer_proyecto_django

para activarlo
source venv/bin/activate

instalar
pip install Django

verificar la instalación
pip show Django

Cambiar de carpeta a la del proyecto

Ejecutar el proyecto
docker-compose up --build

Abrir el navegador en 
http://localhost:8001/admin

e ingresar las credenciales como administrador
